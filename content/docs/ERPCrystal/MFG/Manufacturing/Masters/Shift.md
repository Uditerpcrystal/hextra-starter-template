## Introduction

Captures shift timing of the factory / office.

#### Mandatory Fields

|Mandatory Fields|  
  |:------|
  | Shift Code <br> - Auto generated by System
  | Shift Name
  | Shift Hours <br> - Pre selected as 08(08, 12)
  | Shift Start Time
  | Shift End Time
